# Dataset

This project uses the **CIC-DDoS 2019** dataset.

## Download Instructions

1. Visit: https://www.unb.ca/cic/datasets/ddos-2019.html
2. Download the training and testing CSV files
3. Place them in this `data/` folder:
   - `training_dataset_CIC_DDoS_2019.csv`
   - `testing_dataset_CIC_DDoS_2019.csv`

## Note
Dataset files are not included in this repository due to size constraints.
